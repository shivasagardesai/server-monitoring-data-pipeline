# server-monitoring-data-pipeline
Data pipeline and Power BI dashboard for monitoring server performance.

Data trasformation steps performed:

• Loaded the dataset using Python (Pandas).
• Checked for missing values in the dataset.
• Checked and removed duplicate records.
• Converted the timestamp column into datetime format.
• Validated CPU and memory utilization values to ensure they are within acceptable ranges.
• Created additional analytical features such as Total Network Traffic, Availability Rate, and Server Health Score.
